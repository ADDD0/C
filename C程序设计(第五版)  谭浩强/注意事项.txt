除必要的空格外，不含其他空格
大括号要空两格
if，else，for，while等有缩进时空两格，语句除单语句外均空四格，例：
while ((c = getchar()) != EOF)
  putchar(c);    //空两格
for (n = 0; (c = getchar()) != EOF; n++)
  {              //空两格
    putchar(c);  //空四格
    if (n > 10)  //空四格
      break;     //空六格
  }              //空两格

注释使用//
注释离左侧语句尾至少两空格
除函数注释、程序注释外均左对齐
