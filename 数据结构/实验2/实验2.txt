        int ListDel_Sq(SqList *L, int i, int k) {
            //顺序表L,从第i个元素起连续删除k个元素.
            ......
        }  //本算法可参考教材24页算法2.5
    (2)扩充上边主函数(main)对初始顺序表L(含n个数)的调用
        ListDel_Sq(L, 2, 3)函数,表示从L中第二个元素起连续删除3个数,并输出调用ListDel_Sq后的L中的元素的值.
        提示:ListDel_Sq这个函数如果是高效的,那么它的时间复杂度应为O(n),如果时间复杂度为O(n*n),那么需要对算法进行改进.
